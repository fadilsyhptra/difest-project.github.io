# 🌍 Earthquake Data Center (Website)

This project is a web-based application that displays recent earthquake data from around the world using a public API.

Built for the **DEFEST competition**, this project successfully achieved **3rd place 🥉**.

---

## 📌 Overview
The website provides real-time and historical earthquake information, allowing users to explore seismic activity globally. Data is fetched dynamically from an external API and presented in a clean and interactive interface.

---

## 🚀 Features
- 🌐 Display global earthquake data  
- 📡 Fetch real-time data from API  
- 📊 Interactive and responsive UI  
- ⚡ Fast and lightweight performance  

---

## 🛠️ Technologies Used
- JavaScript (Node.js)  
- Tailwind CSS  
- External Earthquake API
